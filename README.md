# NotTooShabby Theme for Zed

Pretty much fine theme for Zed.

## Installation

1. Open Zed
2. Press `Cmd+Shift+P` (or `Ctrl+Shift+P` on Linux/Windows)
3. Type `zed: extensions`
4. Search for "NotTooShabby"
5. Click Install
6. Press `Cmd+K Cmd+T` (or `Ctrl+K Ctrl+T`) to open theme selector
7. Select "NotTooShabby"

## Features

- Dark background optimized for long coding sessions
- Carefully selected colors for Ruby/Rails development
- High contrast for better readability
- Muted git blame text that doesn't distract
- Soft purple for instance variables
